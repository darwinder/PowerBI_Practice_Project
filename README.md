# PowerBI EcomExpress Dashboard
* This project features an interactive Power BI dashboard to analyse sales, product performance and overall business trends for EcomExpress. The dataset covers the period from 1 jan 24, to 6 nov 24.
* This dashboard was created to expaND MY KNOWLEDGE ON POWER BI AND was created as a part of my learning process for power bi, and is based on a youtube tutorial by creator xyz, and used the dataset provided by them.
* i recreated the dashboard shown in the video and made small amendments like choosing a different colour theme and an additional visual.
* this interactive dashboard analysses overall ecom express business performance. and includes visuals like revenue trend overtime, category contributions, and is interactive based on categories included in slicer on top. this also included key kpis like total revenue, average order value (which is high due to laptops and all) and cancellation rate to support business decision making.
